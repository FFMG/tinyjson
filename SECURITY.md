# Security Policy

## Supported Versions

Only the latest version of TinyJSON is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within TinyJSON, please do **not** open a public issue. Instead, please report it privately.

You can contact the maintainer directly:
- **Florent Guelfucci**: [Insert Contact Method/Email if public, or instruction to contact via GitHub Profile]

Please include:
- A description of the vulnerability.
- Steps to reproduce (including sample JSON if applicable).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
