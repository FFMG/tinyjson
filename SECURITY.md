# Security Policy

## Supported Versions

Only the latest version of TinyJSON is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| < 0.2   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within TinyJSON, please do **not** open a public issue. Instead, please report it privately.

You can contact the maintainer directly:
- **Florent Guelfucci**: Please visit my [GitHub Profile](https://github.com/FFMG) and use the contact information provided there (e.g., social links or email if visible) or open a private "Security Advisory" if available on the repository.

Please include:
- A description of the vulnerability.
- Steps to reproduce (including sample JSON if applicable).
- Potential impact.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
